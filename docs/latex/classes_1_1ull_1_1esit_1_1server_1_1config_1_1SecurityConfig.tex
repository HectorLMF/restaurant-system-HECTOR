\doxysubsection{Detailed Description}
Spring Security configuration for the backend. 

Defines\+:
\begin{DoxyItemize}
\item Password encoder used for hashing passwords.
\item HTTP endpoint security policies\+: all endpoints are currently open (no authentication required).
\end{DoxyItemize}

The login logic is handled manually in the Auth\+Controller.
